# 安装 iproute2



## MAC

```bash
brew install iproute2mac
```

> https://github.com/brona/iproute2mac
>
> - 没有提供 `ss` 的实现 [ss command not found?](https://github.com/brona/iproute2mac/issues/20)



## Linux

新版本一般都自带

```bash
yum install -y iproute2
```
